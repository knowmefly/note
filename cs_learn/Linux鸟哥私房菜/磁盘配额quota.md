# quota
1. inode：限制“文件数量”
2. block：管理用户磁盘容量
- **quota**
	- 管理用户资源，可以针对用户和组

# Software RAID
1. RAID0：性能好
2. RAID1：完整备份
3. RAID 0+1，RAID 1+0：上述组合
4. RAID 5：用一块硬盘保证数据被完整备份
5. spare disk：预备磁盘
6. mdadm：软件模拟RAID
