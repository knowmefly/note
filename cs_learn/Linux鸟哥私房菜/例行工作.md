- **at**
	- /etc/atd.allow:允许任务
	- /etc/atd.deny:不允许用户
	- 一次性命令
	- atq：查看at任务
	- atrm：删除at任务
- **batch**
	- 在负载小于0.8的时候启动任务，否则等待何时机会
- **crontab**
	- -e:修改循环任务
	- -l:列出循环任务
	- -r:删除所有任务

- **run-parts**
	- 后跟目录可允许目录下所有可执行script脚本
